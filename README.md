## Github Repository
https://github.com/prashidi/udagram

## Local deployment
1. npm i
2. npm run dev
3. eb init 
4. Modify the .elasticbeanstalk/config.yml file to include deploy:
  artifact: ./www/Archive.zip
5. npm run build
6. eb create 
7. eb deploy

## EB endpoint
